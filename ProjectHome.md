This is a Theme Builder for ExtJS framework.

It allows to construct a theme and to unload it.

It is based on displacement of colours or other css properties of theme's elements.
The elements are
Base Color,
Header Color,
Background color,
Border color,
Header Font,
Font,
Window transparency,
Toolset (Default, Vista, TargetProcess, Graphite).

Theme builder for ExtJS now supports next ExtJS versions:
3.3, 3.2, 3.1, 3.0, 2.3.
